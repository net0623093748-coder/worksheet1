# แบบฝึกหัด Git

ชื่อ-นามสกุล : นาย สุภกิจ มั่นคง

ห้อง : 2/2

เลขที่ : 12

----------------------------------

ข้อที่ 1
Git คืออะไร

ตอบ : ระบบควบคุมเวอร์ชัน (Version Control System)

----------------------------------

ข้อที่ 2
Repository คืออะไร

ตอบ :  คลังเก็บข้อมูล

----------------------------------

ข้อที่ 3
คำสั่งใดใช้ Clone Repository

ตอบ : git clone ตามด้วย URL

----------------------------------

ข้อที่ 4
คำสั่งใดใช้เพิ่มไฟล์เข้าสู่ Staging Area

ตอบ : git add

----------------------------------

ข้อที่ 5
คำสั่งใดใช้บันทึกการเปลี่ยนแปลง

ตอบ : git commit -m "ข้อความอธิบายการเปลี่ยนแปลง"

----------------------------------

ข้อที่ 6
คำสั่งใดใช้ส่งข้อมูลขึ้น GitHub

ตอบ : git push -u origin main

----------------------------------

ข้อที่ 7
วันนี้ได้เรียนรู้อะไรเกี่ยวกับ Git

ตอบ : คำสั่งพื้นฐาน พื้นที่การทำงานของ Git

----------------------------------
